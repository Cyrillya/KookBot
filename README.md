<h1 align="center">Kook Bot</h1>

<div align="center">

一个Kook机器人模组，在泰拉瑞亚服务器和Kook频道之间进行消息同步

</div>

## ✨ 功能
1. Kook频道和游戏服务器内聊天信息互传
2. (计划中) 通过机器人指令查询服务器状态、玩家状态等
